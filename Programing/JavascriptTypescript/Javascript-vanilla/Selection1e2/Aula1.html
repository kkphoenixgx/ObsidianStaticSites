Date

No js, para trabalhar com tempo, tem uma classe chamada de Date que tem seus métodos e funções sobre o tempo.

com Date.now() por exemplo dá para pegar o timeStamp e se você não se lembra o que é o time stamp é os milissegundos contados dês de a criação da linguagem c que no js tem +3 casas do que nas outras.

Date:

  • x = new Date() Dá para pegar a data formatada;
  • x.getDate = pega a data de hoje
  • x.getFullYear = pega o ano com 4 dígitos
  • x.getMonth e month começa em 0 = Pega o mês
  • x.toLocaleDateString(“x”) = pega a data do formato local
